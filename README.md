# AWS Data Engineering and ML Labs

This repository provides a unified collection of **hands-on labs, datasets, and tutorial resources** for:

- **AWS Certified Data Engineer – Associate (DEA-C01)**
- **AWS Certified Machine Learning – Associate (MLA-C01)**
- **AWS Certified Machine Learning – Specialty (MLS-C01)**

The labs in this repository support the learning journey of readers using the associated KnoDAX study guides and YouTube tutorials.
